How to use the manual build flow

Go to the **Dashboard** panel, then locate the "Quick Trigger" container block. Click the manual deploy button. The system will look for changes, bundle the app directory setup, and spin up an environment.

Frequently Asked Questions

Are variables secure?

Yes, variable structures are stored inside decoupled runtime variables within isolated spaces.

How many active sandboxes can I build?

The platform supports up to five simultaneous builds by default.
